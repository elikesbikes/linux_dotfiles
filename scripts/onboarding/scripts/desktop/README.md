# Desktop Category

## Purpose
Graphical applications for daily use.

## Structure
This category is split by install mechanism:

### Flatpak apps
Installed and updated via Flathub.

### Native apps
Installed via apt due to system or hardware access needs.

- Clapgrep

- install_desktop_native_apps.sh
