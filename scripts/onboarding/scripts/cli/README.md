# CLI

CLI installs daily-use command-line tools.

## Tools installed
- Neovim (APT, Omakub-style)
- Starship prompt (APT)
- Direnv
- Stow
- Fastfetch
- Exa
- Zoxide
- Figlet
- Yazi (Snap)

## Notes
- No `apt update` is executed in this category
- Starship is activated via dotfiles, not the installer
- `tree-sitter-cli` is explicitly removed after Neovim install
